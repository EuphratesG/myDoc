# 搜广推相关知识
短期消费指标  
·点击率=点击次数/曝光次数
·点赞率=点赞次数/点击次数
·收藏率=收藏次数/点击次数
·转发率=转发次数点击次数
·阅读完成率=滑动到底次数/点击次数×f（笔记长度）

只关注短期消费指标是不对的，衡量推荐系统的好坏更重要的是北极星指标  
→·用户规模：
·日活用户数(DAU)、月活用户数(MAU). 和推荐系统好坏强相关  
·消费：
·人均使用推荐的时长、人均阅读笔记的数量。
·发布：发布激励通常由冷启动负责
·发布渗透率、人均发布量。
